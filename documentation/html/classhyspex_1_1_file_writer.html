Detailed Description

Class for writing .hyspex files.

This class uses the pImpl idiom to reduce external dependencies. We also want to avoid relying on STL for maximum compatibility. This class is not thread-safe.

This class is intended to be used if you need more flexibility than Recorder provides. It is also used internally by Recorder.
